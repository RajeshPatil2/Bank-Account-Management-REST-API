PROJECT : BANK ACCOUNT MANAGEMENT REST API
========================================

TECHNOLOGY STACK
----------------
**Java 17**
**Spring Boot**
**Spring Data JPA**
**RESTful Web Services**
**MySQL Database**
**Maven**
**Exception Handling**

PROJECT OVERVIEW
----------------
This project is a **Spring Boot REST API** developed for **Bank Account Management**.
It allows users to **create accounts**, **view account details**, **deposit money**,
and **delete accounts** using secure REST endpoints.

FEATURES
--------
• **Create Bank Account**
• **Get Account By ID**
• **View All Accounts**
• **Deposit Amount**
• **Delete Account**
• **Custom Exception Handling**
• **Global Exception Handler**

REAL-TIME USE CASE
------------------
Used in **Banking and Finance Applications** to manage
**customer accounts**, **balance updates**, and
**secure transaction processing**.

EXCEPTION HANDLING
------------------
• **AccountNotFoundException**
• **InvalidAmountException**
• **Centralized Global Exception Handler**
